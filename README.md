# How to INSTALL
- Clone this git
- Disable any antivirus
- Copy any file to folder game location
- Now you can enable antivirus
- When you done everything now you can play a game

## How to UPDATE this mod
- Disable any antivirus
- Pull this git
- Copy any file to folder game location
- Now you can enable antivirus
- When you done everything now you can play a game
